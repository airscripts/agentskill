# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.1.x   | yes       |

## Reporting a Vulnerability

If you discover a security issue, please report it privately before opening a
public issue.

Preferred reporting path:

- Email: <francesco@airscript.it>

If you are unsure whether something is a security issue, you can still report
it with reproduction details, affected versions, and potential impact.
